# 🍽 Food Safety Compliance Tracking System

## 📌 Project Overview
The Food Safety Compliance Tracking System is a DBMS-based project that analyzes food inspection data. It helps monitor whether food establishments follow safety regulations by tracking inspection results and identifying failures.

---

## 🎯 Objectives
- Track food safety inspections  
- Identify pass and fail cases  
- Analyze inspection trends over time  
- Visualize data using graphs  

---

## 🛠 Technologies Used
- Python  
- Pandas  
- Matplotlib  
- CSV (as database)
